Exercício GitHub

Este é um exercício para praticar o uso do GitHub. Vamos criar um repositório, fazer commits e aprender a colaborar com outros desenvolvedores.

Passos para o exercício:

  1. Crie um repositório no GitHub.
  2. Clone o repositório para o seu computador.
  3. Crie um arquivo README.md e adicione uma descrição do projeto.
  4. Faça um commit com a mensagem "Adiciona README.md".
  5. Envie as alterações para o repositório remoto.
